# Web Admin Backend - 管理后台

这是一个简单的Web管理后台系统，支持用户管理、内容管理和数据统计可视化。

## 功能特性

### B1: 后台框架与用户列表
- ✅ 简洁的管理界面（使用现代化设计）
- ✅ 用户列表查看
- ✅ 用户删除功能

### B2: 内容管理与图表
- ✅ 帖子/内容列表查看
- ✅ 帖子删除功能
- ✅ ECharts数据可视化
  - 📊 月度帖子统计柱状图
  - 🥧 用户活跃度饼图

## 技术栈

- **后端**: Node.js + Express
- **前端**: HTML5 + CSS3 + JavaScript (Vanilla)
- **图表库**: ECharts 5.x
- **数据存储**: 内存存储（演示用）

## 安装与运行

### 1. 安装依赖

```bash
npm install
```

### 2. 启动服务器

```bash
npm start
```

### 3. 访问系统

打开浏览器访问: http://localhost:3000

## API 接口

### 用户管理
- `GET /api/users` - 获取所有用户
- `DELETE /api/users/:id` - 删除指定用户

### 内容管理
- `GET /api/posts` - 获取所有帖子
- `DELETE /api/posts/:id` - 删除指定帖子

### 统计数据
- `GET /api/statistics` - 获取统计数据（用于图表）

## 功能说明

### Dashboard（仪表盘）
- 显示系统总体统计数据
- 月度帖子数量柱状图
- 用户活跃度分布饼图

### 用户管理
- 查看所有注册用户列表
- 支持删除用户操作
- 删除后自动刷新列表和统计数据

### 内容管理
- 查看所有发布的帖子
- 支持删除不适当内容
- 删除后自动更新相关数据
